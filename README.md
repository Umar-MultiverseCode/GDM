## 🚀 Deployment & Installation

### 1. Repository Setup
```bash
git clone https://github.com/clinical-ai-intelligence/GDM-Precision.git
cd GDM-Precision
```

### 2. Environment Configuration
```bash
# Initialize isolated environment
python -m venv venv
source venv/bin/activate  # Mac/Linux
.\venv\Scripts\activate   # Windows

# Dependency injection
pip install -r requirements.txt
```

### 3. Execution
```bash
# Initialize model telemetry
python train_model.py

# Launch portal
streamlit run app.py
```

## 🧪 Clinical Diagnostic Parameters

The engine evaluates 8 critical biomarkers for autonomous risk assessment:

1.  **Pregnancies**: Maternal parity history.
2.  **Glucose**: Post-challenge plasma glucose concentration.
3.  **Blood Pressure**: Diastolic arterial pressure (mm Hg).
4.  **Skin Thickness**: Triceps skin fold thickness (mm).
5.  **Insulin**: 2-Hour serum insulin (mu U/ml).
6.  **BMI**: Mass-to-height ratio (kg/m²).
7.  **Diabetes Function**: Hereditary risk coefficient.
8.  **Age**: Clinical patient age.
